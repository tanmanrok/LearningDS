## ⚠️ Important Caveats

1. **Data Leakage Warning:** `unique_login_days` feature derives from same engagement data defining adoption—performance inflated in this historical context. For production, use lagged features (first-week logins only).

2. **Historical Data:** Dataset spans 2012-2014. Should validate findings on recent user cohorts via A/B testing.

3. **Missing Context:** Analysis lacks product usage metrics, user demographics, competitive data. These could refine predictions.

4. **Causation vs. Correlation:** Findings show correlation. Marketing opt-in may signal intent rather than cause adoption.
